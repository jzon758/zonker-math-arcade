# Project Instructions

- This is a static HTML/CSS/JS math arcade hosted on GitHub Pages.
- Prefer small, targeted edits.
- Do not rewrite entire files unless explicitly asked.
- Do not print full files unless explicitly asked.
- Show diffs and concise summaries.
- Preserve existing games and behavior unless the task specifically requires changes.
- For new games, follow the existing arcade structure:
  - Add a page in `games/`.
  - Add game logic in `scripts/`.
  - Link the game from `index.html` and `about.html` if appropriate.
  - Add only necessary shared styles to `style.css`.
- Before editing, inspect relevant files and give a brief plan when the task is nontrivial.
- After editing, check for broken links, missing script references, missing IDs/classes, and obvious JavaScript syntax errors.
- Do not install packages, delete files, push to GitHub, or edit files outside this project without asking first.
