# New Arcade Game Prompt

Use this prompt when asking Codex to help add a new game to Zonker Math Arcade.

## Game Details

- **Game name:** [Fill in the game name]
- **Skill focus:** [Fill in the math skill or concept]
- **Grade/level:** [Fill in the target grade or difficulty level]
- **Core gameplay idea:** [Describe what the player does moment to moment]
- **Controls:** [Keyboard, mouse, touch, buttons, or other controls]
- **Scoring:** [How points are earned or lost]
- **Win/loss or time rules:** [Describe how a round ends, whether there is a timer, lives, misses, streaks, or levels]
- **Files likely needed:** [Usually a page in `games/`, game logic in `scripts/`, links in `index.html` and `about.html`, and limited shared styles in `style.css`]
- **What existing behavior to preserve:** [List any games, pages, styles, or mechanics that should not change]

## Default Instructions

- Inspect relevant files first.
- Do not edit yet until a plan is approved.
- Keep changes small.
- Follow `AGENTS.md`.
- Do not print full files.
- Do not push to GitHub unless asked.
